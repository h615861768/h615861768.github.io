自适应

发表于 2016-05-26

概念：

自适应是一种可以适应各种不同屏幕分辨率的概念，不需要为每种不同分辨率的手机单独适配，适合手机端

适用：

但是如果要用于手机与PC ，就不适合了，因为元素放大与缩小在手机跟pc端的体验相差还是挺大的

项目适用场景

pc建议使用响应式布局过渡到平板
手机建议使用自适应布局.
有很多动态效果的pc网站不适合做响应式.

ajax

post:

webstorm本地服务器环境配置

发表于 2016-02-06

webstorm 如何链接本地服务器

  • 步骤一：
    运用服务器软件wamp开启服务器

  • 步骤二：
    打开webstorm软件
    进入设置 –> Deployment

  • 步骤三：
    新增一个server,type选择local or mounted folder

  • 步骤四：
    修改 Web server root URL: 为 http://localhost:xxx xxx为你服务器的端口号

  • 步骤五：
    点击open测试一下看跳转是否正常
    成功会显示一个index of/ 你服务器软件下www目录中的所有文件

    不成功，可能是端口号被占用，可以尝试修改其他端口号。
