适配解决

问题出现:各种设备虽然你设置了视口,但是宽度还是不一致,每种设备会出现偏差
解决的方法有以下的几种

1.百分比适配

实现思想:

宽度可以设置百分比,例如25%
高度一般使用具体的值,高度无法百分比;
一般是配合其他布局使用,页面适配性太低,一般不主要进行使用

2.viewport适配

不设置视口手机的宽度是980px;
实现的原理:
不操作device
动态去设置缩放的比例
使每个设备都使用固定320;
在页面操作的时候就可以使用宽度为320px进行写样式

<script>
(function(){
    var w = window.screen.width;
    var targetW = 320;
    var scale =  w/targetW ;  // 当前尺寸/目标尺寸
    var meta = document.createElement("meta")    ;
    meta.name = "viewport";
    meta.content = "user-scalable = no,initial-scale="+scale+ ",minimum-scale="+scale + ",maximum-scale="+scale;
    document.head.appendChild(meta);
})();
</script>

3.rem适配

rem / em 不相等
em是根据字所在的父级的字体大小来进行设置的

rem的 r = root 跟节点
rem是根据根节点的字体大小来进行设置的

方法:

(function(){
    var html = document.documentElement;
    var hWidth = html.getBoundingClientRect().width;
    html.style.fontSize = hWidth / 16 + "px";


})();

这个时候16rem == 满横屏
4rem == 4分之1

ps: 假如页面是750px的设计图
750px / 15 fontSize 750px分成了15份
每份是50px 计算一个元素的rem宽度 等于 元素宽度/50 等于rem

<body>
    16rem 
</body>
