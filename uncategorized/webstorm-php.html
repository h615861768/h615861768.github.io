webstorm 如何链接本地服务器

  • 步骤一：
    运用服务器软件wamp开启服务器

  • 步骤二：
    打开webstorm软件
    进入设置 –> Deployment

  • 步骤三：
    新增一个server,type选择local or mounted folder

  • 步骤四：
    修改 Web server root URL: 为 http://localhost:xxx xxx为你服务器的端口号

  • 步骤五：
    点击open测试一下看跳转是否正常
    成功会显示一个index of/ 你服务器软件下www目录中的所有文件

    不成功，可能是端口号被占用，可以尝试修改其他端口号。
