1.TypeScript概述

TypeScript 是由 C#语言之父 Anders Hejlsberg 主导开发的变成语言，本质上是向JavaScript语言添加可选的静态类型和基于类的面向对象变成，同时也支持诸如接口、命名空间、装饰器等特性，它相当于是JavaScript的超集。

2009年开始设计的TypeScript语言，经历了几年的发展后，最终向ECMAScript靠拢，实现了其标准，主要有类型校验、接口、装饰器等特性，使代码更加的规范化，有利于项目的维护。

安装

浏览器要运行TypeScript程序，需要先编译成浏览器能识别的代码，可以通过自带的tsc编译器来编译TypeScript文件。

charles乱码

发表于 2017-04-03

charles 青花瓷抓包乱码的问题

安装

不介绍，百度一大把。

初始设置

  1. 需要安装相关证书
    3.10及以下版本是在 http://www.charlesproxy.com/ssl.zip 处进行下载证书

    如今都是4.0+版本，所以安装方式不是这样的

    安装方法：

    1. 进入charles菜单【help】 --- 【SSL Proxying】 ---【install charles Root Certificate】
2. 证书添加到mac的钥匙串里面了，但是默认是不被信任的，需要设置
3. 双击该证书，展开【信任】- 【加密套接字协议层（SSL）】 点击设置成始终信任

  2. mac上必要的设置
    1.在charles菜单 【Proxy】 – 【Proxy Settings】,勾选Enable transparent HTTP proxying.
    2.【Proxy】 –【SSL Proxying Settings】 host 填* port填写443

  3. 获取本机的IP
    方法1：在clarles菜单【Help】 – 【Local IP Address】 点击弹出就可以看到
    方法2：在mac的【系统偏好设置】– 【网络】选择你所连接的网络即可看到
    方法3：在终端里面输入 ifconfig en0 inet后面跟着的就是ip

  4. 在手机端，连接与mac同一wifi，在同一个网络段。打开手机wifi高级设置勾选高级设置代理主机名填写mac的ip地址 代理服务器填写【Proxy Settings】中的8888端口号

  1. 然后就可以开心的抓包了。

参考地址 http://itangqi.me/2016/05/30/tips-for-using-charles/
